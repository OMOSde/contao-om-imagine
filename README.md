# Contao-OM-Imagine

A symfony bundle for the Open Source PHP Content Management System Contao. It allow to manipulate uploaded images.

## System requirements

 * Contao 4.4+

## Installation

You can install the module through composer.

```
composer require omosde/contao-om-imagine
```
